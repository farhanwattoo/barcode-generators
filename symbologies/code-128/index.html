コード 128: Modulo-103 検証によるサブセット A、B、および C

Code 128 は、幅広い文字セットをサポートしながら、比較的狭いフットプリントに多くの情報を詰め込むため、物流向けの主力線形シンボルです。実装の重要な詳細は、Code 128 が実際には 1 つの規格内に存在する 3 つのエンコード モードであるということです。サブセット A、サブセット B、サブセット C です。コード 128 を 1 つの未分化のアルファベットとして扱うジェネレーターは通常、密度を表に残します。

サブセット A は大文字と制御コード用に最適化されており、サブセット B はほとんどのビジネス システムが必要とする印刷可能な ASCII 範囲をカバーし、サブセット C はシンボルごとに 2 桁をエンコードすることで数値データを圧縮します。このサブセット C の動作により、エンコーダーがモードをインテリジェントに切り替えると、SSCC フラグメント、日付、出荷 ID などの長い数値文字列が大幅に短くなります。

サブセットこんな方に最適注意事項
A大文字 + 制御文字印刷不可能なコントロールまたはスキャナー コマンドが関係する場合に便利です。
B混合ビジネステキスト名前、SKU、人間向けのペイロードの共通のデフォルト。
C密な数値文字列最大の圧縮を実現するために、シンボルごとに 2 桁をエンコードします。

チェックサムの動作

有効な Code 128 シンボルはすべて Modulo-103 チェック ディジットで終わります。エンコーダーは、選択された開始コードの数値から開始し、後続の各シンボルにその位置の重みを乗算し、その結果を合計し、103 で割ったときの剰余を取得します。ページまたは API が、サブセットの切り替え後にチェック デジットを再計算せずにコード 128 を出力すると主張する場合、それは信頼できるエンコーダーではありません。

実施ガイダンス

  • 印刷スケーリング中にクワイエット ゾーンとモジュール幅が正確に保たれるように、ベクトル出力を優先します。
  • 両側に少なくとも 10 × X のクワイエット ゾーンを予約します。
  • 長い偶数長の数値が出現するたびに、サブセット C に切り替えます。
  • ロゴ、ラベル、または BWR の変更後のスキャナーの動作を検証します。

このシンボルが仕事に適しているかどうかを判断する方法

コード 128: Modulo-103 検証を使用したサブセット A、B、および C は、ペイロードに必要な文字セット、利用可能な水平または正方形のスペースの量、既に導入されているスキャナー、およびシンボルが小売業者、GS1、または業界固有の標準を満たす必要があるかどうかという 4 つの運用上の質問にマッピングすると、最も簡単に評価できます。この決定フレームにより、より適切な技術的適合性が存在する場合に、チームが使い慣れたコード タイプを選択することがなくなります。

  • ペイロード ルールから始めます。数値のみ、混合テキスト、バイナリ データ、または構造化識別子は、それぞれ異なるシンボルを指します。
  • クワイエット ゾーン、倍率、スキャナーの距離が選択プロセスの一部となるように、物理的な表面を早めにチェックしてください。
  • ユースケースが小売 POS、社内物流、部品の直接マーキング、または顧客対応のモバイル スキャンであるかどうかを確認します。
  • ジオメトリやコントラストが不十分な場合、技術的に正しいシンボルでも失敗する可能性があるため、エンコーダ検証と印刷検証を組み合わせてください。

検証に関する一般的な質問

Code 128 のサブセット、密度の最適化、エンタープライズ ラベルの Modulo-103 チェックサムの動作について詳しく説明します。プロダクションアートワークを承認する前に、正確な文字数制限、チェックサムまたはエラー修正の動作、スキャナーの互換性、および人間が判読できる書式設定ルールを確認してください。これらのチェックにより、チームは、選択したバーコードがソフトウェアでは機能するが、スキャンする必要がある実際のチャネルでは機能しないことをプロセスの後半で発見することを防ぎます。

FAQ

同じペイロードでもコード 128 がコード 39 より短いのはなぜですか?

Code 128 にはより高密度のエンコード モデルがあり、サブセット C を使用して数値データをペアのシンボルに圧縮できるためです。

自動モードの代わりにサブセット B を強制する必要があるのはどのような場合ですか?

ペイロードが印刷可能な混合テキストであることがわかっていて、デバッグまたは検証のために予測可能なシンボル構造が必要な場合は、サブセット B を強制します。
