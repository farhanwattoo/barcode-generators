Generating a technically correct barcode is only half the job. If you download it in the wrong file format or at insufficient resolution, the printed result will be blurry, pixelated, or compressed — and the barcode will fail to scan. This guide explains the differences between SVG, PNG, and PDF barcode formats and tells you exactly which to use for your situation.

The three barcode file formats: SVG, PNG, PDF

SVG

Scalable Vector Graphics

Mathematical description of shapes. Infinitely scalable — perfect at any size from a 1cm label to a billboard.

✓ Best for print ✓ No quality loss at any size ✓ Smallest file size ✗ Not supported by all apps
PNG

Portable Network Graphics

Pixel-based image with lossless compression. Quality depends on the resolution it was generated at.

✓ Universal compatibility ✓ Lossless — no compression artefacts ✓ Supports transparency ✗ Pixelates if scaled up
PDF

Portable Document Format

Print-ready document format. May contain vector or raster barcode depending on the generator.

✓ Ready to print directly ✓ Preserves exact dimensions ✗ Harder to embed in other designs ✗ Quality depends on source

SVG barcodes — the best choice for professional print

SVG (Scalable Vector Graphics) is a vector format, which means the barcode is described as mathematical shapes rather than pixels. A vector barcode has perfectly sharp, clean edges at any size — whether you print it at 2cm on a label or 2 metres on a banner. There is no pixelation, no blurring, no artefacts.

SVG is the correct choice for product packaging artwork supplied to a printer or packaging manufacturer, label design files used in Adobe Illustrator, Affinity Designer, or InDesign, any situation where the barcode may be scaled to different sizes, and retail packaging where GS1 print quality standards must be met.

Industry recommendation: Always supply SVG barcodes to packaging designers, label manufacturers, and print houses. A vector file guarantees the barcode remains crisp regardless of how it is reproduced.

How to use an SVG barcode in design software

All professional design tools support SVG. In Adobe Illustrator, use File → Place and select your SVG file. In Affinity Designer, drag and drop the SVG directly onto your canvas. In Canva, upload the SVG as a custom element. The barcode will be fully scalable and editable as a vector object.

SVG barcodes on the web

SVG barcodes can also be embedded directly in HTML pages using an img tag or inline SVG code. They render sharply on all screen densities including retina and high-DPI displays, and their file sizes are typically smaller than equivalent PNG files.

PNG barcodes — best for digital use and documents

PNG (Portable Network Graphics) is a raster format — the image is made of pixels arranged in a grid. PNG uses lossless compression, meaning no image data is discarded when the file is saved, unlike JPEG. This makes PNG safe for barcodes as long as the image is generated at sufficient resolution.

PNG is the right choice for embedding barcodes in Word documents, Excel spreadsheets, and Google Docs, adding barcodes to emails and presentations, displaying barcodes on websites and apps, and barcode images that will be used at a fixed, known size.

PNG resolution requirements

The quality of a PNG barcode depends entirely on the resolution it was generated at. The minimum safe resolution for printing is 300 DPI. At this resolution, a UPC-A barcode at 100% GS1 nominal size (37.29mm x 25.91mm) should be at least 440 x 306 pixels. For Code 128 or other high-density barcodes with narrow bars, generate at 600 DPI to ensure bar edges are crisp.

Never scale up a PNG barcode. If you generated a barcode at 150 DPI and then enlarge it in Word or Photoshop, the image becomes pixelated. The bars soften, lose their sharp edges, and the barcode will fail to scan. Always generate the PNG at the resolution you need from the start, or use SVG instead.

PDF barcodes — best for direct printing

A PDF barcode file is a print-ready document containing your barcode at a defined physical size. PDF is the most convenient format when you want to print barcodes directly from your computer without needing to import them into a design application. Simply open the PDF and print at 100% scale — no resizing, no scaling, exact dimensions guaranteed.

PDF barcodes are useful for quick single-use printing without a design tool, sending barcodes to non-designers who need to print them immediately, and workflows that require a fixed-size print-ready file. A vector PDF (where the barcode is described as paths, not pixels) is equivalent in quality to SVG. Our barcode generator outputs vector PDFs for all 1D barcode types.

Why you should never use JPEG for barcodes

JPEG is a lossy compression format — it deliberately discards image data to reduce file size. This compression process introduces artefacts along high-contrast edges, which is exactly what barcode bars are. The sharp boundary between a black bar and a white space becomes blurred and smeared by JPEG compression, making bar widths inaccurate. A barcode scanner reads bar widths to decode data, so blurred edges mean incorrect decoding or outright scan failure.

Never use JPEG for barcodes under any circumstances. Always use SVG, PNG, or PDF. If someone sends you a JPEG barcode, reject it and request a PNG or SVG replacement.

DPI requirements for barcode printing

DPI (dots per inch) determines how many ink dots your printer lays down per inch of paper. The more dots per inch, the finer the detail the printer can reproduce.

Thermal label printers and DPI

Thermal label printers — the type used in warehouses, retail stockrooms, and shipping departments — are rated by their print head resolution. Common ratings are 203 DPI, 300 DPI, and 600 DPI. A 203 DPI thermal printer is adequate for large Code 128 shipping barcodes on standard label stock. For small barcodes, Data Matrix codes, or labels that must meet GS1 verification grades, a 300 DPI or 600 DPI thermal printer is recommended.

Frequently asked questions

What is the best file format for a barcode?
SVG is the best format for professional print because it is a vector format that scales to any size without pixelation. PNG at 300 DPI or higher is best for digital use, documents, and web display. PDF is convenient for direct printing workflows.
What DPI do I need for a printed barcode?
A minimum of 300 DPI for reliable scanning. For high-density barcodes with narrow bars use 600 DPI. Never print a barcode below 200 DPI.
Can I use a JPEG barcode for printing?
No. JPEG lossy compression blurs barcode bar edges, causing scan failures. Always use SVG, PNG, or PDF for barcodes.
Can I scale up a PNG barcode?
No. Scaling up a raster PNG introduces pixelation and blurs bar edges. Generate the PNG at the exact resolution you need, or use SVG which scales without quality loss.
What format should I send to a label printer?
Send SVG for the highest quality. If your label manufacturer requires a PDF, request confirmation that they accept vector PDFs. If only raster formats are accepted, supply PNG at 600 DPI or higher.
