バーコード印刷キャリブレーション ガイド: ハードウェアのスキャン可能性と寸法精度の最大化【2026】

画面上では完璧に見えるバーコードでも、産業用ハードウェアで印刷するとひどく失敗する可能性があります。このガイドでは、欠陥ゼロのチェックリスト、一般的なサーマル プリンタの実践的なキャリブレーション手順、および DPI 構成全体で安全な X 寸法を計算するためのツールを紹介します。内蔵コンバータを使用してプリンタの DPI を正しいモジュール幅に合わせ、高額なチャージバックを回避します。

コアコントラストルール: カラーウェーブ理論とハードウェアレーザー吸収

赤色レーザー スキャナは 630 ～ 670nm 付近で動作します。表面の色によっては、この光を吸収せずに反射する場合があります。スキャナーのモデルを確認しない限り、赤/オレンジ色の背景や光沢のある金属フィルムは避けてください。

ベクター (SVG/EPS) とラスター (PNG/JPG) の制作形式

ベクトルのエクスポートでは、正確なバーのジオメトリが保証されます。ラスター画像にはアンチエイリアスが発生する可能性があります。ラスターを使用する必要がある場合は、十分に高い DPI でエクスポートし、可能な場合はアンチエイリアスを無効にしてください。

いつどれを使うか

  • SVG/EPS/PDF: 印刷ネイティブのワークフローや、正確なジオメトリが必要な場合に使用します。
  • PNG: 高 DPI でエクスポートした場合、消費者向けのラベルや小規模な印刷に使用できます。
  • JPEG: バーコードの場合は避けてください。非可逆圧縮によりエッジの忠実度が損なわれます。

産業用サーマルプリンター (Zebra、Dymo、Brother) のキャリブレーション手順

  1. サーマルヘッドとローラーを清掃してください。
  2. 複数の送り速度で濃度調整プリントを実行します。
  3. 暗さを少しずつ調整します。 BWR を測定し、反復します。
  4. ISO/IEC 15416 検証器または代表的なレーザー スキャナーで検証します。

バー幅縮小 (BWR): いつ、どのように行うか

BWR は、多孔質基材上でのインクの広がりを補正します。フレキソ印刷および熱転写の実行では -5% ～ -10% の BWR で開始し、その後マイクロメーター検証ステップで実際のバーを測定し、繰り返します。

制作チーム向けのよくある質問

Code 128 バーコードを印刷するのに安全な最小サイズはどれくらいですか?

それは選択した X 次元によって異なります。一般的な 203 DPI プリンタの場合、安全な X は 10 ～ 12 ミルです。検証ツールで検証します。

透明または金属フィルム基材にバーコードを印刷できますか?

はい、ただしスキャン動作はスキャナーによって異なります。代表的なハードウェアでテストしてください。金属表面の場合は、白い下層でコントラストを高めるか、レーザー スキャナを避けてください。

携帯電話のカメラではバーコードがスキャンされるのに、市販のハードウェアでは失敗するのはなぜですか?

携帯電話のカメラは CMOS センサーと画像補正を使用します。市販のレーザー銃は反射光の遷移に依存しており、コントラストや波長の不一致に対してより敏感です。
