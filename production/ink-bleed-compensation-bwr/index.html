インクにじみ補正と BWR: バーが閉まらないようにする

インクが広がると、良好なデジタル ジオメトリが不良な印刷ジオメトリになってしまいます。バー幅縮小 (BWR) は、印刷前にバーをわずかに縮小することで広がりを打ち消し、最終的な物理マークが意図した幅に近づくようにします。

BWR が必要な理由: インクゲインの物理的メカニズム

インクまたはサーマル トナーが基板に接触すると、毛細管現象と材料の吸収によって意図した境界の外に広がります。紙やコートされていないボードなどの多孔質素材では、X 次元の 5〜15% に相当するゲインが発生することがあります。光沢フィルムや熱転写ポリエステルでは広がりは少ないものの、ヘッドの圧力によってトナーが縁に流れることがあります。フレキソ印刷では、プレートのショアー硬度と印刷速度がゲインの大きな変数です。

ISO/IEC 15416 検証規格は、各バーおよびスペース幅の公差を名目上の X 次元の特定の割合以内に規定しています。BWR はそのマージンを事前に確保するためのツールです。補正なしで実行すると、高速フレキソ ラインはグレード C を下回るリスクがあり、小売業者や物流チャネルで使用されるエリア イメージャーやレーザー スキャナーでの読み取り失敗率が急増します。

印刷プロセス別 BWR 開始値の目安

印刷プロセス推奨 BWR 開始値備考
フレキソ（コート紙）−5〜−8%プレート硬度とインク粘度で変動
フレキソ（段ボール）−8〜−15%表面の粗さと吸収で幅広い調整が必要
熱転写（ポリエステル）−2〜−4%ヘッド圧による流れに注意
熱転写（紙ラベル）−4〜−7%リボン種別（wax / wax-resin / resin）によって異なる
ダイレクトサーマル−3〜−6%ヘッドの暗さ設定と密接に連動
デジタルオフセット0〜−3%インクゲインは比較的小さいが検証は必須

これらは出発点です。実際の基板と実際の印刷ハードウェアで測定したバー幅を使用して反復的に調整してください。プレス設定だけを見て「補正済み」と判断するのは危険です。

BWR 適用ワークフロー: 5 ステップ手順

  1. 補正なしで基準ラベルを印刷し、較正済みバーコード検証器またはマイクロメーターでバー幅を実測します。
  2. 測定した平均バー幅と名目上の X 次元の差を算出します。例: 名目 10 mil に対して 11.2 mil 実測 → ゲイン +12%。
  3. アートワーク システムまたはバーコード生成ソフトに −10〜−12% の BWR を入力して再印刷します。SVG ワークフローでは各バー矩形の幅を縮小し、スペース幅を等量だけ拡大してシンボル総幅を維持します。
  4. 補正後サンプルを再実測し、ISO/IEC 15416 検証器でグレードを確認します。グレード B 以上を目標とし、主要小売チャネル向けはグレード A を目指します。
  5. 速度・ロール温度・素材・リボンが変更されるたびに手順 1 から繰り返します。

バーが「閉じる」症状と診断

バーが閉じるとは、隣接するバーとスペースが一体化してしまう障害です。検証器は「エッジ コントラスト シグナル (ECR)」の低下または「アパーチャーの不一致」として報告します。目視では太いバーが隣接スペースに流れ込んでいるように見えます。

  • フレキソの場合: プレートが劣化した場合やインクが過供給されている場合によく発生します。インク量を減らし、プレートの摩耗状況を確認します。
  • サーマルの場合: 暗さ設定が高すぎる、またはヘッドが汚れているとバーが閉じます。ヘッドを清掃してから暗さを段階的に下げます。
  • 高温環境: 熱転写では基板の温度上昇によりトナーが広がります。環境温度の変化に合わせてプリンターを再キャリブレーションします。
  • 特定シンボルのみ失敗: 特定の N:W 比（ナロー:ワイド）が現在のゲイン設定と合っていない可能性があります。その記号タイプの名目 X 次元を確認します。

再キャリブレーションが必要なイベント一覧

BWR 設定は一度決めれば終わりではありません。以下のいずれかが発生した場合は必ず再検証してください。

  • 印刷素材（基板・ラベルストック・コーティング）の切り替え
  • リボン種別またはインクロットの変更
  • プリンターのサーマルヘッド交換またはフレキソ プレートの交換
  • 環境湿度または温度の大幅な変化
  • 速度・暗さ設定の変更
  • バーコード生成ソフトまたは RIP システムのアップデート

各再キャリブレーションの記録にはプリンターモデル・素材ロット・BWR 設定値・検証器グレード・担当者名を含めてください。後日の障害調査でプロセス変化に原因を特定するための証跡になります。

エスカレーション判断基準

以下の状況は個人調整の範囲を超えています。品質担当者またはプリンター ベンダーに連絡してください。

  • BWR を最大限適用してもグレード C を下回る場合 — プレート・ヘッド・素材自体の問題の可能性。
  • 同じ設定で複数シフトにわたって結果が不安定な場合 — ヘッドの部分的損傷またはロール圧力の変動を疑います。
  • 特定ロットのみ失敗し他は問題ない場合 — 素材ロット間の吸収特性の差異を調査します。
