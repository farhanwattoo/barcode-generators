始める前に: 適切なバーコードの種類を選択してください

バーコードを作成する際の最も重要な決定は、正しいシンボルを選択することです。間違ったバーコード タイプ (小売 POS スキャン用の製品の QR コードなど) を使用すると、目的のシステムでバーコードを読み取ることができなくなります。

オンラインでバーコードを作成する方法 — 5 つのステップ

オンラインでのバーコードの作成には 1 分もかかりません。ソフトウェアをインストールする必要はありません。

1

無料のバーコードジェネレーターを開く

に行く barcode-generators.com/barcode-generator.html。アカウントやサインアップは必要ありません。このツールは完全にブラウザ内で実行されます。

2

バーコードの種類を選択してください

ドロップダウンからシンボルを選択します: UPC-A、EAN-13、Code 128、QR コード、データ マトリックス、またはその他のタイプ。不明な場合は、上の表を参照するか、 バーコードの種類ガイド.

3

データを入力してください

エンコードする値を入力します。 UPC-A の場合、これは 11 桁の数値です (ジェネレータは 12 番目のチェック デジットを自動的に計算します)。コード 128 の場合は、任意の英数字文字列。 QR コードの場合は、URL、電話番号、電子メール、または最大 4,296 文字のテキスト。

4

サイズ、色、形式のオプションを設定する

出力形式を選択します: プロフェッショナルな印刷用の SVG (無限に拡張可能)、デジタル用途またはドキュメント用の PNG、または直接印刷用の PDF。バーの色 (暗い必要があります) と背景の色 (明るい必要があります) を設定します。赤いバーは使用しないでください。赤は、ほとんどの 1D スキャナで使用される赤色レーザー光を吸収しません。

5

生成、プレビュー、ダウンロード

「生成」をクリックします。画面上でバーコードをプレビューします。バーの下にある人間が判読できる数字が目的のデータと一致していることを確認します。ファイルをダウンロードします。大規模に使用する前に、物理的なバーコード スキャナとスマートフォンのカメラの両方でテストスキャンして、正しく読み取れることを確認します。

拡大印刷する前に必ずテストしてください。 10,000 枚のラベルを印刷する前に、生成したバーコードを物理スキャナーまたはスマートフォンでスキャンします。チェック ディジット エラーまたは間違ったシンボル体系があると、すべてのラベルがスキャンできなくなります。

小売バーコード (UPC-A または EAN-13) を作成する方法

小売バーコードは、エンコードする製品番号が世界的に一意であり、正式に登録されている必要があるため、他のタイプよりも複雑です。

ステップ 1 — GS1 会社プレフィックスを取得する

小売販売用の正規の UPC-A または EAN-13 バーコードを作成するには、まず GS1 (バーコード番号を世界的に管理する非営利団体) から GS1 企業プレフィックスを取得する必要があります。 GS1 では、会社のプレフィックスに対して年間料金がかかります。 gs1.org またはあなたの国の GS1 会員組織にアクセスして登録してください。個々の GTIN を販売するサードパーティの再販業者からバーコードを購入しないでください。大手小売業者は GS1 データベースをチェックし、GTIN がブランドに登録されていない製品を拒否する場合があります。

ステップ 2 — 各製品に GTIN を割り当てる

プレフィックスを取得したら、各製品に一意の商品参照番号を割り当てます (サイズ、色、バリエーションごとに独自の番号が必要です)。会社のプレフィックスと品目参照を組み合わせて、GTIN-12 (UPC-A の場合) または GTIN-13 (EAN-13 の場合) を形成します。

ステップ 3 — バーコード画像の生成

GTIN を入力してください UPC-A バーコード ジェネレーター または EAN-13 バーコードジェネレーター。このツールはチェック デジットを計算し、バーコードをレンダリングします。アートワーク ファイルをパッケージ化するために SVG としてダウンロードします。

ステップ 4 — 最小サイズ要件を確認する

UPC-A バーコードの公称サイズは、倍率 100% で 37.29 mm x 25.91 mm です。 GS1 仕様では、このサイズの 80% から 200% の範囲が許容されます。 80% 未満の印刷を行うと、POS でスキャンが失敗する危険があります。最終的に印刷されたバーコードが GS1 の印刷品質要件を満たしていることを必ず確認してください。

配送用バーコード (コード 128) を作成する方法

配送用バーコードは、当局に番号を登録する必要がなく、自分でデータを定義できるため、小売用バーコードよりも簡単です。

  1. どのデータをエンコードするかを決定します: 注文番号、追跡番号、シリアル番号、または GS1 アプリケーション ID を使用した組み合わせ (大手小売店に発送する場合)
  2. を開きます Code 128 バーコード ジェネレーター
  3. 英数字データを入力します。 Code 128 は文字、数字、記号を処理します
  4. SVG または高解像度 PNG としてダウンロード
  5. 配送ラベルに人間が読めるテキストバージョンと並べて貼り付けます
大手小売店への発送は可能ですか？ Walmart、Target、Amazon、または同様の大規模小売店に発送する場合、SSCC、GTIN、バッチ番号、有効期限をエンコードする特定のアプリケーション識別子を含む GS1-128 形式 (プレーン コード 128 ではない) が必要になる可能性があります。ラベルを作成する前に、小売店の案内ガイドを確認してください。

QRコードの作成方法

QR コードは、任意のテキスト、URL、またはデータが有効であるため、作成するのが最も簡単なバーコード タイプであり、登録や特別な番号付けは必要ありません。

QR コードでエンコードできるもの

  • URL — Web サイト、ランディング ページ、または製品ページへのリンク
  • プレーンテキスト — 最大 4,000 文字までのメッセージまたは情報
  • 電子メール — スキャン時にアドレス指定済みの電子メールの下書きを開きます
  • SMS — 事前に宛先が指定されたテキストメッセージを開きます
  • 電話番号 — スキャンされると通話を開始します
  • WiFi認証情報 — スキャナーのデバイスを WiFi ネットワークに自動的に接続します
  • vカード — 連絡先カードをスキャナーのアドレス帳に保存します

誤り訂正レベル

QR コードには 4 つのエラー修正レベルがあります。L (許容誤差 7%)、M (15%、適切なデフォルト)、Q (25%、パッケージングに適しています)、および H (30%、QR コード内にロゴを追加する場合に必要)。レベルが高くなると、コードの損傷耐性が高まりますが、データ容量は減少します。

QRコードの最小サイズ

スマートフォンを確実にスキャンするには、QR コードを印刷する際のサイズは少なくとも 2cm x 2cm (約 0.8 インチ) である必要があります。看板や大判印刷の場合は、スキャン距離に基づいて比例的に拡大縮小します。

Excelでバーコードを作成する方法

Excel スプレッドシートにバーコードを追加するには 2 つの方法があります。バーコード フォントの方法は Code 39 で機能します。無料の Code 39 フォントをインストールし、アスタリスクで囲まれたセル (="*"&A1&"*") にデータを入力し、そのセルにバーコード フォントを適用します。 EAN-13 および UPC-A バーコードの場合、より信頼性の高い方法は、オンライン ツールを使用してバーコードを生成し、PNG としてダウンロードし、[挿入] → [画像] で画像として挿入することです。

Wordでバーコードを作成する方法

同じ 2 つのアプローチが Microsoft Word にも適用されます。1 つは単純な Code 39 ラベルのバーコード フォント方法、もう 1 つはバーコードを外部で生成し、[挿入] → [図] を使用して画像として挿入する方法です。 Word でサイズを変更する場合は、常に縦横比を維持してください。バーコードを水平方向に引き伸ばさないでください。水平方向に引き伸ばすとバーの幅が変わり、スキャンできなくなります。

印刷とサイズの要件

正しく生成されたバーコードでも、印刷が正しくないとスキャンに失敗する可能性があります。印刷されたバーコードを常にスキャンできるようにするには、次のルールに従ってください。

解決

最低でも印刷してください 300 DPI。非常に小さいバーコード、または幅の狭いバーコード (高密度の Code 128 など) の場合は、600 DPI 以上を使用します。印刷には、バーのエッジをぼかす圧縮アーティファクトを引き起こす JPEG ではなく、常に SVG または PDF ソース ファイルを使用してください。

色

バーは暗く、背景は明るくなければなりません。赤いバー (レーザー スキャナーの波長が間違っている)、明るい色のバー、暗い背景は避けてください。白地に黒のバーは最も信頼性の高い組み合わせです。

クワイエットゾーン

すべてのバーコードには、両側にクワイエット ゾーンと呼ばれる空白スペースが必要です。 UPC-A および EAN-13 の場合、これは公称サイズで両側で少なくとも 2.31 mm です。パッケージのアートワーク、テキスト、またはグラフィックがクワイエット ゾーンに侵入しないようにしてください。これは、小売パッケージのバーコード スキャンが失敗する最も一般的な原因の 1 つです。

バーコードを作成する準備はできましたか?

無料のオンラインジェネレーター。 UPC-A、EAN-13、コード 128、QR コード、データ マトリックス、PDF417、および 15 以上。 PNG、SVG、または PDF としてすぐにダウンロードできます。

よくある質問

無料でバーコードを作成するにはどうすればよいですか?
無料のオンライン バーコード ジェネレーターを使用します。バーコードのタイプを選択し、データを入力し、「生成」をクリックして、PNG、SVG、または PDF としてダウンロードします。ソフトウェアやアカウントは必要ありません。
バーコードにはどのようなデータを入力すればよいですか?
UPC/EAN 小売バーコードの場合は、製品の登録済み GTIN をエンコードします。 Code 128 配送バーコードの場合は、注文番号または追跡番号をエンコードします。 QR コードの場合は、URL、連絡先カード、WiFi の詳細、または任意のテキストをエンコードします。
店舗で販売するために商品のバーコードを作成できますか?
はい。ただし、小売バーコード (UPC-A または EAN-13) の場合は、法的に認められた GTIN を取得するために、最初に GS1 企業プレフィックスを登録する必要があります。有効な GTIN を取得したら、オンライン ツールを使用してバーコード イメージを無料で生成します。
バーコードにはどのファイル形式を使用すればよいですか?
プロフェッショナルな印刷用の SVG (ピクセル化せずに任意のサイズに拡大縮小)。 300 DPI のデジタル使用およびドキュメント用の PNG。直接印刷用のPDF。 JPEG は決して使用しないでください。
バーコードが正しいかどうかをテストするにはどうすればよいですか?
生成されたバーコードをスマートフォンのカメラでスキャンし、デコードされた値が目的のデータと一致することを確認します。小売バーコードの場合は、GTIN が GS1 データベースに登録されていることも確認してください。
